# Project Documentation

Tài liệu này bao gồm toàn bộ thông tin cần thiết để hiểu, cài đặt, vận hành và phát triển tiếp dự án Document Manager.

Các tài liệu chính:
- `docs/SRS.md` - Yêu cầu phần mềm và chức năng chi tiết.
- `docs/API_SPEC.md` - API endpoints và định dạng request/response.
- `docs/DEPLOYMENT.md` - Hướng dẫn cài đặt và chạy môi trường phát triển.
- `docs/ARCHITECTURE.md` - Kiến trúc hệ thống, mô hình dữ liệu và luồng xử lý.
- `docs/USE_CASES.md` - Use case, luồng use case và sơ đồ.
- `docs/ERD.md` - ERD chi tiết.
- `docs/USER_GUIDE.md` - Hướng dẫn người dùng cuối các chức năng chính.
